## 📐 Google Sheets の物理構成

### **推奨設定**

| 項目 | 設定値 | 理由 |
|------|--------|------|
| **用紙サイズ** | A4 横向き | テーブルが見やすい |
| **フォント** | Noto Sans JP, 11pt | 日本語対応、読みやすい |
| **セルの高さ** | テキストエリアは自動折り返し | 複数行入力に対応 |
| **列幅** | A列: 150, B-F列: 120-150 | バランスの取れた幅 |
| **背景色** | セクションごとに異なる色 | 視認性向上 |
| **枠線** | 全セルに枠線 | 表として統一感 |

## 🔧 Google Sheets での実装手順

### **Step 1: 新規スプレッドシート作成**
```
Google Drive → 「新規」→「Google Sheets」→ スプレッドシート作成
シート名を「実験記録テンプレート」に変更
```

### **Step 2: 基本情報の入力**
```
A1: "記録ID" → B1: =TEXT(NOW(),"YYYYMMDD")&"-"&SEQUENCE(1)
A2: "作成日時" → B2: =TODAY()
```

### **Step 3: セクション見出しの追加**
```
A4: "【セクション1】試験の基本情報" （背景色: 濃い青、テキスト色: 白）
A20: "【セクション2】試験の実施体制" （同上）
...
```

### **Step 4: 説明欄の追加**
```
各項目の右隣（C列）に説明欄を追加
例：B4「試験概要」→ C4「この試験の背景・概要を簡潔に説明（100-200字）」
フォント: 10pt, イタリック, グレー色 (#999999)
```

### **Step 5: 試験結果テーブルの実装**
```
A57:F57 にテーブル見出しを作成
A58以降: 測定項目行を10行分用意（後から追加可能）
テーブル見出しの背景色: 濃い色、テキスト色: 白
```

### **Step 6: 共有設定**
```
Google Sheets の「共有」→ リンク共有（全員が編集可能）
ユーザーが「このテンプレートをコピーして使用」できるようにする
```

## 📝 使用方法

### **新規実験の場合**
```
1. テンプレートをコピー
   「ファイル」→「コピーを作成」
   
2. ファイル名を変更
   「実験記録_20260505_スマートロック耐久試験」
   
3. 各項目を記入していく
   上から順に埋めていけば、自動的に「実験記録」が完成する
```

### **複数の試験を記録する場合**
```
Sheet1（テンプレート）
→ Sheet2: 性能試験_20260505
→ Sheet3: 耐久試験_20260510
→ Sheet4: 動作試験_20260512

各シートはテンプレートをコピーして、試験ごとに別シートとして管理
```

## 🎯 このテンプレートの特徴

✅ **自動生成**: 記録IDと作成日時は自動入力  
✅ **セクション化**: 視覚的に分かりやすい構成  
✅ **説明欄（ヒント）**: 各項目に何を書くべきか明確  
✅ **テーブル形式**: 試験結果を構造化して保存  
✅ **複数シート対応**: 複数の試験を1つのファイルで管理  
✅ **共有機能**: チーム内での共有・確認が容易  
✅ **印刷対応**: A4横向きで印刷可能
