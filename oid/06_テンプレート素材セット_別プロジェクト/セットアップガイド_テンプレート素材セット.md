# セットアップガイド - 営業AI Agent テンプレート素材セット

**目安時間**: 30～60 分（スキップ可能なステップあり）  
**技術難度**: ★☆☆☆☆（ゼロ）  
**必要なもの**: Google アカウント、Claude API キー

---

## 🎯 何をするのか

このガイドでは、以下の 5 ステップで営業自動化システムを構築します：

1. **セットアップウィザード実行** → Gmail ラベル・Google Sheets 自動生成
2. **Claude API キー設定** → AI 連携
3. **テスト実行** → 動作確認
4. **プロンプトカスタマイズ** → 自社向けに調整
5. **本番運用開始** → 自動化開始

---

## Step 1: セットアップウィザード実行（5分）

### 1-1. Google Apps Script を開く

```
https://script.google.com
```

ブラウザで上記 URL にアクセスしてください。

### 1-2. 新規プロジェクト作成

左上の「新規プロジェクト」をクリック：

```
プロジェクト名: 営業AI_Agent_セットアップ
```

### 1-3. セットアップウィザードコードをコピペ

1. 本テンプレートセットの `GAS_セットアップウィザード_全自動.gs` をテキストエディタで開く
2. 全内容をコピー（Ctrl+A → Ctrl+C）
3. Google Apps Script に戻り、デフォルトコード（`Code.gs`）を選択
4. 全て削除（Ctrl+A → Delete）
5. セットアップウィザードのコードをペースト（Ctrl+V）
6. 保存（Ctrl+S）

### 1-4. セットアップウィザード実行

Google Apps Script 画面上部のドロップダウンから `runFullSetup` を選択：

```
関数メニュー: runFullSetup ← ここを選択
▶️ 実行ボタンをクリック
```

### 1-5. 認可を許可

初回実行時に、Google アカウントへのアクセス許可が求められます：

```
「Google Apps Script が Google アカウントへのアクセスをリクエストしています」
   ↓
「許可」をクリック
```

### 1-6. 実行ログを確認

Google Apps Script 画面下部の「実行ログ」に以下のような出力が見えます：

```
🚀 セットアップウィザード開始...

【Step 1】Gmail ラベル作成中...
  ✅ ラベル 'Agent-Inbox' を作成しました
  ✅ ラベル 'Customer-Conversation' を作成しました
  ✅ ラベル '応答済み' を作成しました

【Step 2】Google Drive フォルダ作成中...
  ✅ フォルダ '収益会社_AI_System' を作成しました

【Step 3】Google Sheets テンプレート生成中...
  ✅ CRM-Master シート作成
  ✅ リード管理 シート作成
  ✅ Agent-Log シート作成
  ✅ KPI-Dashboard シート作成
  ✅ Claude-Prompts シート作成
  ...

【Step 4】環境変数自動設定中...
  ✅ SPREADSHEET_ID を設定しました
  
🎉 セットアップ完了！全 8 個のシートが生成されました。
```

✅ **ここまでで Step 1 完了です。**

## Step 2: Claude API キーを設定（2分）

### 2-1. Claude API キーを取得

以下のサイトにアクセス：

```
https://console.anthropic.com/
```

ログインして、API キーを新規作成（または既存キーをコピー）

### 2-2. Google Apps Script に設定

Google Apps Script に戻り、`GAS_Agent1_完全実装コード.gs` を開く：

```javascript
// 最上部に以下の行があります：
const CLAUDE_API_KEY = "sk-ant-xxxxxxx...";
```

`sk-ant-xxxxx` の部分を、Step 2-1 で取得した API キーで置き換える：

```javascript
const CLAUDE_API_KEY = "sk-ant-YOUR_KEY_HERE";
```

保存（Ctrl+S）

✅ **ここまでで Step 2 完了です。**

---

## Step 3: テスト実行（3分）

### 3-1. テストメールを送信

自分のメールアドレスに対して、テストメールを送信：

```
To: あなたのメールアドレス
Subject: [TEST] AI Agent テスト
Body: こんにちは。このメールに自動で返信できるか確認しています。
```

Gmail で送信してください。

### 3-2. 自動応答を確認

1～2 分待つと、自動返信メールが届きます：

```
From: あなたのメールアドレス
Subject: Re: [TEST] AI Agent テスト
Body: （AI が自動生成した返信メール）
```

✅ **自動返信が来たら Step 3 成功です。**

#### 3-3. エラーが出た場合

「実行ログ」でエラーメッセージを確認：

```
❌ エラー: [Error] unauthorized
→ 対処: Claude API キーが間違っています。Step 2-2 を確認してください。

❌ エラー: [Error] ラベル 'Agent-Inbox' が見つかりません
→ 対処: Step 1 が完了していません。セットアップウィザードを実行してください。
```

詳細は「トラブルシューティング.md」を参照。

## Step 4: プロンプトをカスタマイズ（10～20分）

### 4-1. Claude プロンプトライブラリを開く

Step 1 で自動生成された Google Sheets を開く：

```
Google Drive > 収益会社_AI_System > Claude-Prompts シート
```

### 4-2. 業界別プロンプトを選択

プロンプトライブラリに 6 パターンが用意されています：

| 番号 | パターン | 用途 | 対応業界 |
|------|---------|------|---------|
| 1 | 初期接触メール（型A） | 新規リードへの初回メール | IT・SaaS |
| 2 | 初期接触メール（型B） | 新規リードへの初回メール | 建設・重機 |
| 3 | 初期接触メール（型C） | 新規リードへの初回メール | 医療・福祉 |
| 4 | フォローアップメール | 初回接触後 3 日目 | 全業界共通 |
| 5 | リード分類プロンプト | 関心度レベル判定 | 全業界共通 |
| 6 | 次のアクション提案 | 対応履歴から次手を提案 | 全業界共通 |

## Step 5: 本番運用開始（即座）

### 5-1. Google Apps Script で定期実行を設定

`GAS_Agent1_完全実装コード.gs` を Google Apps Script で開き、以下の行を確認：

```javascript
function checkAndReplyToEmails() {
  // メイン関数
}
```

### 5-2. 定期実行トリガーを設定

Google Apps Script 左メニュー：

```
トリガー（⏰アイコン）を開く
   ↓
新規トリガー追加
   ↓
関数: checkAndReplyToEmails
実行イベント: 時間主導型 → 時間タイマー → 1 時間ごと
```

これにより、毎時間自動でメールをチェック・返信します。

### 5-3. 実運用で注意すべき点

**メール送信前の最終確認:**

```
❌ やってはいけないこと：
   - テストなしで本番営業メール自動化を有効化
   - プロンプトを修正しないまま使用

✅ やるべきこと：
   - 初日は「自動応答」機能のみ有効化
   - 3 日間は手動で AI 返信メールを確認
   - 問題がなければ 4 日目から本番運用
```

### 5-4. 最初の 1 週間のチェックリスト

- [ ] Day 1: テストメール 5 件で精度確認
- [ ] Day 2-3: AI 返信メールを手動確認（修正必要な場合は Step 4 でプロンプト調整）
- [ ] Day 4-7: 本番運用開始（毎日ログを確認）
- [ ] Day 8: 1 週間分の成果を Google Sheets KPI ダッシュボードで確認

✅ **ここまでで Step 5 完了です。**
