## よくある質問（FAQ）

### **Q1: 技術がないのですが大丈夫ですか？**
**A:** はい、大丈夫です。全て「コピペして実行するだけ」の設計です。Google アカウントだけあれば、技術知識ゼロで実装できます。

### **Q2: セットアップにどのくらい時間がかかりますか？**
**A:** 目安は 30～60分です。
- セットアップウィザード: 5分
- Claude API キー設定: 2分
- テスト確認: 3分
- プロンプトカスタマイズ: 10～20分

### **Q3: Claude API の費用はいくらですか？**
**A:** 従量課金です。目安：
- メール 50 件/月: ¥100～200
- メール 500 件/月: ¥1,000～2,000
- メール 5,000 件/月: ¥10,000～20,000

基本は月 ¥1,000～5,000 程度の費用で運用できます。

### **Q4: 複数の営業メンバーで使えますか？**
**A:** はい。Google Sheets をチーム共有することで複数ユーザー対応が可能です。詳細はカスタマイズガイドを参照ください。

### **Q5: メール送信の頻度を調整できますか？**
**A:** はい。Google Apps Script の定期実行設定で「毎時間」「毎日」「毎週」から選べます。詳細は実装ガイドを参照。

### **Q6: エラーが出た場合、どうすればいいですか？**
**A:** 付属の「トラブルシューティング.md」に 15 の一般的なエラーと対処法をまとめています。それでもうまくいかない場合、FAQ フォームでお問い合わせください。

### **Q7: どの業界に対応していますか？**
**A:** テンプレートは「業界中立」です。SaaS・IT、建設・重機、医療・福祉、営業商社など、どの業界でも使えます。業界別のカスタマイズ例が付属しています。

### **Q8: 既存の CRM（Salesforce など）と連携できますか？**
**A:** 基本は Google Sheets ベースですが、Zapier を使って Salesforce・HubSpot と連携させることができます。詳細はカスタマイズガイドの「外部ツール連携」を参照。

### **Q9: 自動返信内容をカスタマイズできますか？**
**A:** はい。Claude プロンプトライブラリをカスタマイズすることで、返信内容を自由に変更できます。

### **Q10: サポートはありますか？**
**A:** Premium グレード購入者には 1ヶ月間の Slack サポートがついています。スタンダードグレード購入者は、FAQ や実装ガイドでサポートしています。
