# 実験記録自動化アプリ ユーザーガイド

**対象**: 実験記録・試験記録を効率的に記録したい方

---

## 概要

このアプリは以下の流れで、**手書きの実験記録を自動で定形化** して Google Sheets に保存します：

```
手書き記録の写真 → AI が自動抽出 → Google Sheets に記録
```

---

## 4ステップの流れ

### STEP 1: 画像をアップロード 📸

```
      ┌─────────────────────────┐
      │  📷 画像をアップロード   │
      │                         │
      │  ドラッグ&ドロップ      │
      │  または クリックして選択 │
      └─────────────────────────┘
```

**操作**:
- 画像をドラッグ&ドロップ
- または「📷 画像をアップロード」 をクリックして選択

**対応形式**: PNG, JPG, GIF など（スマートフォンで撮影した写真 OK）

**コツ**:
- 手書き部分が見やすく、光の反射がない写真を選ぶ
- テキストが読める解像度なら OK

---

### STEP 2: 試験種別を選択

```
      ┌─────────────────────────┐
      │  性能試験 / 耐久試験    │
      │  動作試験 / その他      │
      └─────────────────────────┘
```

**操作**:
- 試験の種類に応じて **チップをクリック** して選択
- 複数選択はできません（1つ選択）

**種別の目安**:

| 種別 | 例 |
|------|-----|
| **性能試験** | 消費電力、スペック検証、応答速度 |
| **耐久試験** | 連続動作、落下試験、温度サイクル |
| **動作試験** | 起動確認、機能テスト、インターフェース |
| **その他** | 上記以外 |

---

### STEP 3: AI が自動抽出 🤖

```
      ┌──────────────────────────┐
      │  🔍 解析を実行            │
      │                          │
      │  AI が以下を自動抽出：    │
      │  ✓ 試験日時              │
      │  ✓ 製品名                │
      │  ✓ 型番                  │
      │  ✓ 試験条件              │
      │  ✓ 測定値                │
      │  ✓ 観察事項              │
      │  ✓ AI 考察（自動生成）    │
      └──────────────────────────┘
```

**操作**:
1. 「🔍 解析を実行」 をクリック
2. ローディング（2～5秒）
3. フォームに自動入力される

**自動抽出される項目**:

| 項目 | 例 |
|------|-----|
| **試験日時** | 2026-05-10 14:30 |
| **製品名** | M5 Core S3 |
| **型番** | M5-CSU001 |
| **試験条件** | 温度: 25°C / 湿度: 60% |
| **測定値** | 電圧: 5.01V / 電流: 0.5A |
| **観察事項** | テスト中の動作状況 |
| **AI 考察** | データから推論された分析内容 |

**✓ すべて手動で修正可能** (AI は補助的なもの)

---

### STEP 3.5: 確認・修正 ✏️

```
      ┌──────────────────────────┐
      │  内容を確認・修正         │
      │                          │
      │  ✓ 形式が合致しているか  │
      │  ✓ 数値に誤字がないか    │
      │  ✓ 不足情報を追加        │
      │  ✓ 間違いを修正          │
      └──────────────────────────┘
```

**修正方法**:
- 各フォームをクリックして編集
- 複数行は **改行区切り** (Enter キー)

**例: 試験条件の入力**:
```
温度: 25°C
湿度: 60%
気圧: 1013 hPa
```

**例: 測定値の入力**:
```
電圧: 5.01 V
電流: 0.5 A
周波数: 50 Hz
```

---

### STEP 4: 保存完了 💾

```
      ┌──────────────────────────┐
      │  💾 記録を保存            │
      │                          │
      │  Google Sheets に送信     │
      │  ✓ 完了                  │
      └──────────────────────────┘
```

**操作**:
1. 「💾 記録を保存」 をクリック
2. インターネット接続を確認
3. 通知が表示される（2秒）
4. フォームが自動リセット

---

## テキスト入力モード

**手書きメモがない場合や、テキストで直接入力したい場合**:

```
☑ テキストで直接入力する  ← チェック
```

**この場合**:
- 画像は不要（スキップ）
- フォームに手動入力
- AI 自動抽出はなし
- 「💾 記録を保存」 で直接保存

---

## 履歴機能 📋

**「📋 履歴を表示」 をクリック**:

```
      ┌──────────────────────────┐
      │  📋 保存済み記録          │
      │                          │
      │  M5 Core S3 - 性能試験   │
      │  2026/5/10 14:30         │
      │                          │
      │  Arduino Uno - 動作試験  │
      │  2026/5/10 10:15         │
      │  ...                     │
      └──────────────────────────┘
```

**特徴**:
- 過去 **50 件** まで保存
- **ブラウザのローカルストレージ** に保存
- インターネット接続不要
- ブラウザのデータ削除で消える

## よくある質問

### Q1: AI が読み取れない場合は？

**A**: AI が正確に読み取れないことがあります。その場合：

1. **AI が空欄にした項目を手動入力**
2. または**テキストモードで直接入力**

すべて修正可能なので、完全性より「効率化」を優先してください。

---

### Q2: AI 考察が不正確

**A**: AI（Gemini）の推論は参考程度です。

1. **必ずご自身で確認**
2. **重要な分析は手動で上書き**
3. 慣れてくると精度が向上します

---

### Q3: オフラインで使える？

**A**: 一部機能のみ：

| 機能 | オフライン | 必要な環境 |
|------|----------|----------|
| **テキスト入力** | ✓ OK | なし |
| **フォーム編集** | ✓ OK | なし |
| **AI 自動抽出** | ✗ NG | インターネット |
| **Google Sheets 保存** | ✗ NG | インターネット |
| **履歴表示** | ✓ OK | localStorage |

---

### Q4: 複数人で使える？

**A**: はい。Google Sheets を共有すれば、複数人が記録できます：

1. **Google Sheets の URL を共有**
2. **各自が HTML を開く**（ローカルファイルでOK）
3. **各自が記録をアップロード**
4. **Google Sheets に全員分が集約**

---

### Q5: 記録を削除したい

**A**: 以下の方法があります：

**オプション1: Google Sheets で削除**
- Google Sheets の行を選択 → 削除

**オプション2: 履歴から削除**
- 履歴は **ブラウザの localStorage に保存**
- ブラウザの開発者ツール → Application → localStorage → 削除

## 入力のコツ

### 📸 画像撮影

```
✓ 良い例                    ✗ 悪い例
┌─────────────────┐        ┌─────────────────┐
│  □ □ □ □ □    │        │  □ ■ □ □ ■    │
│  手書きが鮮明    │        │  光反射が多い    │
│  明るい背景      │        │  ノートが傾いた  │
│  正面から撮影    │        │  小さく写った    │
└─────────────────┘        └─────────────────┘
```

### 📝 テキスト入力

```
✓ 良い入力                  ✗ 悪い入力
┌────────────────┐         ┌────────────────┐
│ 温度: 25°C     │         │ Temp 25℃      │
│ 湿度: 60%      │         │ 25℃, 60%, ..  │
│ 気圧: 1013 hPa │         │ 25-60 normal   │
└────────────────┘         └────────────────┘
```

**ポイント**:
- **項目: 値 単位** の形式
- **改行区切り** で複数項目
- 単位は省略しない
- スペースは統一

## 実用例

### 例1: M5 Core S3 テスト

```
📸 ステップ1: テスト記録を撮影
   └─ 温度: 25°C, 消費電力: 500mW などが手書きされた紙

🔍 ステップ2: 解析を実行
   └─ AI が自動抽出

✏️ ステップ3: 確認・修正
   └─ 「電圧: 5.01V」など数値を確認

💾 ステップ4: 保存
   └─ Google Sheets に追加
```

### 例2: チームでの耐久試験

```
👥 複数人が記録
├─ 技術者A: 温度サイクル試験記録
├─ 技術者B: 落下試験記録
└─ 技術者C: 動作確認

📊 Google Sheets に集約
├─ 全テスト結果を一覧表示
├─ グラフを作成
└─ 統計分析
```

## トラブル対処

| 症状 | 対処 |
|------|------|
| 画像が読み込まれない | 別のブラウザを試す / キャッシュをクリア |
| 解析ボタンが反応しない | インターネット接続を確認 / API キー確認 |
| Google Sheets に保存されない | URL を確認 / デプロイ更新を確認 |
| 履歴が表示されない | localStorage をリセット |
