### ステップ2: VBAマクロを設定（初回のみ）

**Excel開発タブでマクロを追加:**
1. 「開発」タブ → 「Visual Basic」をクリック
2. 左側の「Module1」をダブルクリック
3. 以下のVBAコードをコピー＆ペースト

**営業メール自動生成テンプレート用:**
```vba
Sub 営業メール自動生成()
    Dim ws As Worksheet
    Dim companyName As String
    Set ws = ThisWorkbook.Sheets("メール生成")
    companyName = ws.Range("B3").Value
    
    If companyName = "" Then
        MsgBox "企業名を入力してください", vbExclamation
        Exit Sub
    End If
    
    ws.Range("B6").Value = "【自動生成メール】" & vbCrLf & _
        "件名: 【営業】" & companyName & "様の業務効率化提案"
    
    MsgBox "営業メール自動生成完了！", vbInformation
End Sub

Sub KPI計算()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("分析")
    Dim sentCount, replyCount As Double
    
    sentCount = ws.Range("B3").Value
    replyCount = ws.Range("B4").Value
    
    If sentCount > 0 Then
        ws.Range("B5").Value = Format(replyCount / sentCount, "0.0%")
        MsgBox "返信率計算完了: " & Format(replyCount / sentCount, "0.0%")
    End If
End Sub
```

**SNS投稿自動生成テンプレート用:**
```vba
Sub SNS投稿自動生成()
    Dim ws As Worksheet
    Dim dayOfWeek As String
    Set ws = ThisWorkbook.Sheets("投稿生成")
    dayOfWeek = ws.Range("B3").Value
    
    If dayOfWeek = "" Then
        MsgBox "曜日を入力してください", vbExclamation
        Exit Sub
    End If
    
    Select Case dayOfWeek
        Case "月": ws.Range("B6").Value = "【トレンド投稿】"
        Case "火": ws.Range("B6").Value = "【How-To投稿】"
        Case Else: ws.Range("B6").Value = "【一般投稿】"
    End Select
    
    MsgBox "SNS投稿自動生成完了！", vbInformation
End Sub

Sub エンゲージ率計算()
    Dim ws As Worksheet
    Set ws = ThisWorkbook.Sheets("分析")
    Dim likes, followers As Double
    
    likes = ws.Range("B3").Value
    followers = ws.Range("B4").Value
    
    If followers > 0 Then
        ws.Range("B5").Value = Format(likes / followers, "0.00%")
        MsgBox "エンゲージ率計算完了！"
    End If
End Sub
```

4. `Ctrl+S` で保存して、Visual Basicエディタを閉じる
