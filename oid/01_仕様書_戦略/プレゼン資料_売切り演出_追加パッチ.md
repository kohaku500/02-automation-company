## 💡 心理学的な効果

### 1. **希少性（Scarcity）の原則**
```
「100社限定」「あと87社」という数字が、
顧客に『逃したら二度と来ない機会』を感じさせます。
```

### 2. **緊迫感（Urgency）の原則**
```
「6月30日まで」「残り時間」というカウントダウンが、
『今決断すべき』という心理を生み出します。
```

### 3. **FOMO（Fear of Missing Out）**
```
「他の企業がもう導入している」「枠が埋まっていく」
という情報が、『買い遅れたら負ける』という恐怖を生みます。
```

### 4. **永遠割引インセンティブ**
```
「今導入すれば、永遠に割引」という約束が、
『今決断すれば、生涯メリットが続く』という心理的報酬を生みます。
```
