## 1. 単価設定の決定フレームワーク

### 1.1 単価算定式

```
単価 = （年間営業目標額 ÷ 年間成約件数）× マージン率
```

または

```
単価 = 顧客規模（人数） × 顧客予算上限（円/年） ÷ 営業件数ターゲット
```

### 1.2 買切り型 vs サブスクリプション型の判断基準

| 判断要因 | 買切り型に適す | サブスク型に適す |
|---------|------------|------------|
| 顧客ロック期間 | 1～3ヶ月程度 | 6ヶ月以上 |
| 顧客の切替コスト | 低い | 高い |
| 更新・保守の必要性 | 低い | 高い |
| 想定月次収益安定性 | 変動性高 | 安定性高 |
| 初期導入の難度 | 簡易 | 複雑 |

### 1.3 サブスク型の場合のMRR逆算式

```
月商目標（円） = 平均顧客数 × 平均単価（月額） × 継続率
年間MRR（円） = 月商 × 12

逆算例：
目標年MRR ¥12,000,000 
→ 月商目標 ¥1,000,000
→ 平均顧客数 N × 月額単価 ¥M × 継続率 95% = ¥1,000,000
```

## 3. 顧客獲得チャネル別戦略

### 3.1 主要チャネル比較表

| チャネル | 初期投資（円） | 初期成約率 | 成熟期成約率 | 顧客LTV | 推奨期間 |
|--------|------------|---------|---------|--------|--------|
| 営業メール | 0～50万 | 1～3% | 3～5% | 低～中 | 6～12ヶ月 |
| インバウンド営業（コンテンツ） | 50～200万 | 3～5% | 5～10% | 中～高 | 12～24ヶ月 |

| パートナー営業 | 20～100万 | 2～5% | 5～10% | 中 | 6～18ヶ月 |

### 3.2 各チャネルの月次運営コスト・返りの計算式

**営業メール型**
```
初期投資      = ツール費 + 営業代行費（初月）= ¥_____
月次固定費    = ツール費 + 人件費 = ¥_____
送信件数/月   = _____ 件
返信率        = _____% （初期：1～3%）
成約率        = _____% （返信→契約：20～50%）
月間成約数    = 送信件数 × 返信率 × 成約率 = _____ 件

月商          = 月間成約数 × 単価（円）= ¥_____
月次ROI       = （月商 - 月次固定費）÷ 月次固定費 × 100
```

**有料広告型**
```
月次広告費        = ¥_____
月次コンバージョン数 = ¥_____ （CPC × クリック数）
コンバージョン率   = _____% 
月間成約数        = コンバージョン数 × 成約率 = _____ 件

顧客獲得単価（CAC）= 月次広告費 ÷ 月間成約数 = ¥_____

LTV:CAC比率 = 顧客LTV ÷ CAC
  （目安: 3:1以上で事業継続可能、5:1以上で成長可能）

**直営業型**
```

## 4. 月商¥100万達成ファネル（数式のみ）

### 4.1 基本ファネル式

```
【チャネル別の月商構成】

月商 ¥100万 = 以下の複数チャネルの合計

チャネル A : 送信件数 NA × 返信率 PA × 成約率 QA × 単価 RA = 月商 MA
チャネル B : 送信件数 NB × 返信率 PB × 成約率 QB × 単価 RB = 月商 MB
チャネル C : 入札数   NC × 成約率 QC × 単価 RC                = 月商 MC

合計 : MA + MB + MC = ¥1,000,000

目標逆算例：
年間目標 ¥12,000,000 / 12ヶ月 = ¥1,000,000/月

チャネルA（営業メール）: 5,000件 × 2% × 30% × ¥30,000 = ¥900,000
チャネルB（広告）     : 20件 × 50% × ¥5,000           = ¥100,000
────────────────────────────────────
合計月商                                           = ¥1,000,000
```

### 4.2 変数調整による達成パターン

```
【パターン1】: 単価重視
  低い送信数 × 高い成約率 × 高い単価 = ¥1,000,000
  例: N × 5% × Q × ¥R

【パターン2】: 送信数重視
  多い送信数 × 低い成約率 × 低い単価 = ¥1,000,000
  例: N × 2% × Q × ¥R

【パターン3】: バランス型
  中程度の送信数 × 中程度の成約率 × 中程度の単価 = ¥1,000,000
  例: N × 3% × Q × ¥R

自社のパターンを選択:
現在の制約条件:
  - 送信リソース上限 : _____ 件/月
  - 品質管理上の成約率上限 : _____% 
  - 市場相場単価 : ¥_____
```

## 7. 注記

本テンプレートは**業界に依存しない数式とフォーム形式**のみで構成されています。

- 業界特化的なアドバイスはありません
- 具体的な商品・サービス例は含まれていません
- すべて「変数」として自社の条件を代入する仕様です

各項目に数値を入力することで、自社の収益化戦略が自動的に構築されます。
