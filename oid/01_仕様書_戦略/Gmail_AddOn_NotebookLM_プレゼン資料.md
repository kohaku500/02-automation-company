## 📌 Executive Summary（1ページ）

### 問題
営業チームは毎日、大量のメールの中から「買ってくれそうな見込み客」を見分けるのに、平均 **30分/日** を費やしています。多くのメールは興味のない問い合わせで、対応に人手をとられ、本当に買いそうな客への返信が遅れています。

### 解決策
**「AI Agent Sales Assistant」** — Gmail に 1 クリックでインストールするだけで、受け取ったメールから **購入意欲を自動判定**（0～100点スコア）し、購入可能性の高いメール（スコア70以上）を自動仕分けするツールです。さらに低スコアメールには、営業担当者の文体を学習した自動返信を生成・送信します。

### ビジネス機会
- **ターゲット**: 営業チーム 5～50人の企業
- **市場規模**: 日本の中小企業 50万社、SaaS化率 15% = 7万5千社
- **初期価格**: ¥9,800～¥29,800/月（スタンダード・プレミアム）
- **想定 3ヶ月売上**: 60～100件販売 × ¥14,800 平均 = **¥600k～¥1M**

### Why Now
- ChatGPT・Claude 普及により、営業チーも「AI で時間短縮」意欲が高い
- Google Workspace 利用企業が急増（76%が 2024-2026 導入済み）
- 営業DX市場が急成長（年平均 18% 成長、2026～2031）

## 🎯 市場機会と背景

### 営業業界の課題（Today's Problem）

営業チームが毎日直面している 3 つの課題：

#### 課題 1: メール仕分けの時間浪費
```
┌─────────────────────────────────┐
│  営業担当者 A さんの 1 日         │
├─────────────────────────────────┤
│ 朝 9:00  → メール 50 件受信     │
│ 朝 9:30  → 「買いそう」な客分け  │ ← 30分費やす
│ 朝 10:00 → ようやく返信開始      │
│ ...                             │
│ その間、本当の見込み客への返信   │
│ 遅延 → 競合に案件を奪われる      │
└─────────────────────────────────┘
```

**数値**:
- 営業担当者は 1 日 100～200 件のメール受信
- そのうち「本当に買いそう」なのは 3～5 件（2～3%）
- 仕分けに平均 30 分費やされている
- 月間 10 時間 = **年間 120 時間（15 営業日分）の浪費**

#### 課題 2: 購入意欲判定が属人的
- 経験がない営業担当者は「誰が本当に買おうとしているのか」判定できない
- メールを読み間違え、温度感の低い客に時間をかけすぎ
- 逆に高確度の客への対応が雑になる
- **結果**: チーム全体の成約率が 10～20% 低下

#### 課題 3: メール返信品質のばらつき
- 返信が丁寧でない → 客が怒る
- 返信が遅い → 競合に負ける
- 返信を忘れる → 客から電話がくる
- 返信内容がテンプレート → 信頼感が下がる

対象企業数: 約 75,000 社
SAM = 75,000社 × 月額 ¥14,800 
    = **¥111B (1,110 億円/年)**
```

#### SOM（Serviceable Obtainable Market）
```
弊社が年 1 で達成可能な市場シェア:
├─ Year 1（2026）: 500 件導入
├─ Year 2（2027）: 2,000 件導入

SOM = 5,000社 × 月額 ¥14,800 × 12 ヶ月
    = **¥888M (8.88 億円/年)**

## 💡 ソリューション概要

### 製品: AI Agent Sales Assistant

#### What（何をするか）

```
【Before】営業担当者の 1 日
↓
メール 100+ 件受信
  ├─ 何度も同じメールを読み直す
  ├─ 「本当に買いそう」か判定に 30 分
  ├─ 重要なメールを見落とす
  └─ 返信が遅れる → 失注

【After】AI Agent 導入後
↓
メール 100+ 件受信
  ├─ AI が自動判定（3 秒）
  ├─ 「買いそう」なメール → 自動転送（スコア 70+）
  ├─ 「買わなそう」なメール → 自動返信（スコア <70）
  ├─ 営業は重要なメールに集中
  └─ 返信時間 30分 → 3分に短縮
```

#### How（どう動くか）

```
Step 1: 設定（初回のみ 5 分）
┌─────────────────────────────────┐
│ • Gmail AddOn をインストール     │
│ • Claude API キー貼り付け        │
│ • 営業チームの文体を学習させる    │
│ → 環境構築完了                   │
└─────────────────────────────────┘
         ↓

Step 2: 毎日自動で動く
┌─────────────────────────────────┐
│ メール受信 (tanaka.agent@..)   │
│         ↓                       │
│ AI が分析（3秒）               │
│ 「このメール、買う可能性 85%」   │
│         ↓                       │
│ スコア 70+ ?                    │
│ ├─ YES → tanaka@.. に転送      │
│ └─ NO  → 自動返信を送信         │
│         ↓                       │
│ 営業は重要なメールだけ対応      │
└─────────────────────────────────┘
```

#### Why（なぜこれが必要か）

営業の成約率は **メール返信速度 + 対応品質** で決まる。

- 返信が 1 時間遅れるだけで成約率は 20% 低下
- テンプレートみたいな返信は信頼されない
- 重要なメールへの対応が遅れると確実に失注する

→ **AI が自動で対応すれば、営業は本当に大事なメールに時間をかけられる**

### なぜ弊社が勝つのか（Competitive Moat）

#### 1. **超シンプルなセットアップ**
- Google Workspace Marketplace から 1-Click インストール
- 競合は「数日間の実装が必要」→ ユーザーの心理的障壁が高い
- 弊社は「5 分で稼働」→ 導入率が 5 倍以上

#### 2. **AI による文体自動学習**
- 営業の「個人らしさ」を AI が学習して返信を生成
- テンプレートではなく、個人の文体で返信 → 顧客が違和感を感じない
- 競合は「テンプレート返信」→ 信頼度が落ちる

#### 3. **購入意欲スコア**
- Claude API を使った「リアルタイム購入意欲判定」
- スコア 70+ なら優先対応、70- なら自動処理
- 営業が「何に時間をかけるべき」か一目瞭然

#### 4. **日本市場特化**
