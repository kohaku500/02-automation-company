### **Step 4: Google Apps Script セットアップ（10 分）**

1. **Google Apps Script を開く**: https://script.google.com

2. **新規プロジェクト作成**:
   - 「新規プロジェクト」
   - プロジェクト名: `Gmail 自動応答ボット v1.0`

3. **ファイル追加（左メニュー）**:
   ```
   + 新規ファイルを作成
   ① ファイル名: Code
      ファイル種類: .gs
   
   ② ファイル名: Agent2
      ファイル種類: .gs
   
   ③ ファイル名: Agent3
      ファイル種類: .gs
   ```

4. **各ファイルにコードをコピペ**:
   - `Code.gs` ← `GAS_Agent1_完全実装コード.gs` の全内容
   - `Agent2.gs` ← `GAS_Agent2_リード情報抽出.gs` の全内容
   - `Agent3.gs` ← `GAS_Agent3_顧客対応サマリー.gs` の全内容

5. **保存**: Ctrl+S

6. **プロジェクト設定から スクリプト ID をコピー**:
   - 左メニュー → プロジェクト設定
   - スクリプト ID をコピー

---

### **Step 5: 環境変数設定（5 分）**

1. **Google Apps Script エディタ → Project Properties**
   - 左メニュー → Project Properties

2. **スクリプト プロパティを設定**:
   ```
   キー: SPREADSHEET_ID
   値: [KPI-Raw-Data（または任意の Sheet）の ID]
   
   ※ Sheet URL から抽出:
   https://docs.google.com/spreadsheets/d/{SPREADSHEET_ID}/edit#gid=0
                                            ↑ これ
   ```

3. **Slack URL（オプション）**:
   ```
   キー: SLACK_WEBHOOK_URL
   値: https://hooks.slack.com/services/... (後で設定可)
   ```
