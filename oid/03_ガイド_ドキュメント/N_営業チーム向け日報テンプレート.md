# N. 営業チーム向け日報テンプレート

**制作日**: 2026-05-03  
**概要**: 毎日 18:00 に営業担当者が記録する日報テンプレート。営業活動の進捗、チャレンジ、改善案を可視化し、チーム内で共有できる標準フォーマット。ワンマン運用の場合は自分用レビュー記録として使用。

---

## 概要

**目的**: 
- 営業日ごとの活動記録
- チャレンジ・学び・改善案の抽出
- 翌日への施策調整
- 週次 / 月次分析の基礎データ

**提出方法**: 毎日 18:00-19:00 に Google Sheets に入力 / メール送信
**対象**: 営業チーム全員（またはワンマン運用時は自分用）
**保管先**: Google Sheets "Daily-Sales-Report" or Email

## N-4. ワンマン運用時の日報保管方法

### 保管場所

**Option A**: Google Sheets（推奨）
- Sheet 名: "Daily-Sales-Report"
- 毎日 1 行追加（Row = 1 Day）
- メリット: グラフ化・集計が容易、整理しやすい

**Option B**: Google Docs（複数ファイル）
- 月ごとに 1 ファイル作成（例: "June_Daily_Reports"）
- メリット: 記述が自由で詳細な分析が可能

**Option C**: Email（メール送信）
- 自分用アドレスに毎日 18:00 に自動送信
- メリット: スマートフォンで簡単に確認可能

---

### 保管期間

| 期間 | 保管場所 | 用途 |
|---|---|---|
| 当日 | Google Sheets / Docs | リアルタイムで確認・改善施策に反映 |
| 1週間 | Google Sheets | 週次分析・トレンド分析に使用 |
| 1ヶ月 | Google Sheets + PDF 化 | 月次レビュー・来月への改善計画立案 |
| 6ヶ月以上 | Google Drive Archive フォルダ | 中期トレンド分析・年間評価の参考資料 |

## N-5. 日報からの自動アクション

### リスク検知ロジック

日報に以下のいずれかが記載されたら、**即座にアラート実行**:

```
✗ 返信率が前日から -5% 以上低下
✗ AI Agent エラーが 3 件以上
✗ テンプレート改善が「超急務」と記載
✗ メンタル欄で「疲労度 8/10 以上」 or「モチベーション 3/10 以下」
✗ ピボット判定の判定基準を満たした（リスク対応マニュアル参照）
```

**アクション**: リスク対応マニュアルを参照し、対応フロー実行

---

### 施策反映フロー

```
日報完成（18:00）
    ↓
施策 1-3 の内容確認
    ↓
「実装可能か」判定
    ↓
翌日朝 06:00 に優先順位付きで実行
    ↓
テンプレート更新 / セグメント変更 / システム改善 実行
    ↓
2-3 日後に効果測定
```

## N-6. テンプレート使用例（参考）

### 例 1: 返信率が好調な日

```
【DATE: 2026-06-03】

1. 本日の営業活動（数値）
   営業メール: 50通, 返信: 12通, 返信率: 24% ✓ (目標: 10-15%)

2. 本日の成功事例（top 3）
   成功 1: テンプレート v2 で返信率 24% 達成。件名の CTA 強化が奏功。
   成功 2: 朝 09:00 発送グループで返信率が 30% に達成。発送時間帯の最適化が効果的。
   成功 3: 中堅企業（50-500名）セグメントで 28% 返信率。ターゲット層の再確認が効果的。

3. チャレンジ: なし（好調なため特に課題なし）

4. 明日への改善施策
   施策 1: テンプレート v2 を引き続き採用。v3 開発は Day 5 以降に予定。
   施策 2: 朝 09:00 発送を明日も優先（返信率 30% 実績）
   施策 3: 中堅企業セグメント拡大。明日 60 通追加送信予定。

7. リスク判定: すべて OK ✓
```

### 例 2: 返信率が低調な日

```
【DATE: 2026-06-04】

1. 本日の営業活動（数値）
   営業メール: 50通, 返信: 4通, 返信率: 8% ✗ (目標: 10-15%)

2. 成功事例: なし（返信率が低いため、成功パターン抽出できず）

3. チャレンジ（課題）
   課題 1: テンプレート v2 の返信率が前日 24% → 8% に低下。原因: 不明（天候？曜日？）
   課題 2: AI Agent v1 で 2 件のエラー発生。メール送信遅延の可能性。
   課題 3: スタートアップセグメントからの返信がゼロ。ターゲットのズレ？

4. 明日への改善施策
   施策 1: テンプレート v3 を緊急リリース。件名を「問題提起型」から「価値提示型」に変更。
   施策 2: AI Agent エラーを調査。ログファイル確認 & Google Apps Script デバッグ予定。
   施策 3: スタートアップセグメント除外。代わりに大企業向けテンプレートを強化。

7. リスク判定:
   ⚠️ 返信率が目標から -7% 低下 → リスク対応マニュアル「修正判定」を参照して Day 5 に施策調整予定
```
