## ❓ よくある質問

### Q: エラーが出た場合は？

**A**: 実行ログの詳細を確認してください。

一般的なエラー:
```
❌ ラベル作成失敗
  → Gmail に既に同名のラベルが存在する可能性
  → 既存ラベルを削除して再度実行

❌ フォルダ作成失敗
  → Google Drive の容量不足
  → 別のアカウントで試す

❌ Spreadsheet 作成失敗
  → Google Sheets の API が有効になっていない
  → Google Cloud Console で API を有効化
```

---

### Q: セットアップウィザードが不要になった場合は？

**A**: 削除しても問題ありません。

```
セットアップウィザードプロジェクトは、
初回セットアップ時だけ必要です。
その後は削除して構いません。
```
