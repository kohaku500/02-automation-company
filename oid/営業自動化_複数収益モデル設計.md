## Model A：B2B営業自動化SaaS（月額課金型）

### 1. ビジネスモデル概要

**対象顧客**: 中堅企業（従業員 50～300人）の営業・マーケティング部門

**提供内容**:
- AI Agent による営業メール自動生成 + 配信
- リード自動分類・優先度付け
- フォローアップシーケンス自動化
- 営業成績ダッシュボード・レポート

**差別化要素**:
- Claude API による高品質なメール生成
- 業界別・企業規模別テンプレートライブラリ
- 日本語に特化した営業心理学を反映したプロンプト設計

---

```
基本プラン（月額）
├─ ユーザー 1～3名：¥50,000/月
├─ ユーザー 4～10名：¥80,000/月  
└─ ユーザー 11～30名：¥120,000/月

追加オプション
├─ 高度なカスタマイズ：¥+30,000/月
```

**年1ユーザー平均単価**: ¥80,000

---

#### 案2：AI Agent実行数ベース価格
```
月額基本料金：¥30,000

変動料金（AI Agent 実行数）
├─ 0～1,000実行/月：¥0（基本料に含む）
├─ 1,001～5,000実行/月：¥0.05 × 実行数
├─ 5,001～10,000実行/月：¥0.03 × 実行数
└─ 10,001+実行/月：¥0.02 × 実行数

【実行例】
営業メール生成 1日20件 × 20営業日 × 1ヶ月 = 400実行
初期接触メール → フォローアップメール×3 = 1社あたり4実行
リード分類・優先度付け = 1件あたり0.2実行

典型月：月額 ¥30,000 + (3,000実行 × ¥0.05) = ¥30,150
```

**年1ユーザー平均単価**: ¥50,000～80,000

## Model B：営業代行型（成功報酬ハイブリッド）

### 1. ビジネスモデル概要

**対象顧客**: 営業組織を持たない・営業人員不足の中堅企業・スタートアップ

**提供内容**:
- リード探索・リスト作成
- 営業メール代行送信（AI生成）
- 初期接触～ヒアリングまでのプロセス自動化
- 契約成立時：成功報酬

**課金方式**:
1. **月額固定料金**（リード探索・メール代行基本料）
2. **成功報酬**（契約成立時：契約金額の10～15%）

---

### 2. 単価設定

#### パターン1：月額固定 + 成功報酬ハイブリッド
```
月額基本料金：¥30,000
（含まれるもの：リード探索 50社/月、営業メール代行）

成功報酬：契約金額 × 12%
（例）
- 顧客単価 ¥100万の契約成立 → ¥12万の報酬
- 顧客単価 ¥500万の契約成立 → ¥60万の報酬

【想定シナリオ】
営業代行対象企業の平均契約金：¥300万
成功率：25%（リード50社 → 成約12社/年）
月平均成約数：1社/月

月商 = ¥30,000（月額固定） + ¥36,000（成功報酬） = ¥66,000/月
```

#### パターン2：成功報酬オンリー型
```
成功報酬：契約金額 × 15～20%

【利点】
- 顧客の初期投資ゼロ → 購入障壁が極めて低い
- 当社のインセンティブが100%成功に向く

【欠点】
- 月商の変動性が高い
- 初期数ヶ月の収入が不安定

【推奨用途】
- 既に営業基盤がある企業の「追加営業リソース」としての利用
- 高単価案件（¥1M以上）を扱う業界
```

### 1. 優先順位の付け方

**推奨実装順序** (工数・リスク・キャッシュフロー勘案)

```
Week 1-2：Model B の営業代行スタート（即月商効果が見えやすい）
```
